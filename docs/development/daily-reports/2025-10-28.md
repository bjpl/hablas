## Technical Decisions Made

### Decision 1: edge-tts Over Azure TTS

**Context**: Needed high-quality Spanish audio for 59 resources

**Options**:
- **Azure TTS**: Best quality, but complex setup, paid tier required
- **gTTS**: Free, simple, but robotic voice quality
- **edge-tts**: Free, Microsoft quality, simple API

**Decision**: edge-tts

**Rationale**:
- Free (no API key required)
- High quality (uses Microsoft's voices)
- Simple Python implementation
- Fast batch processing
- Colombian Spanish support

**Trade-offs**: Requires internet connection for generation (acceptable for build-time use)

### Decision 2: Serif Typography

**Context**: Body text readability for language learners

**Before**: Sans-serif (system fonts)

**After**: Serif fonts for content, sans-serif for UI

**Rationale**:
- Better readability for long-form content
- Professional, elegant appearance
- Distinguishes content from UI
- Improves learning focus

### Decision 3: Remove Gradients

**Context**: Mobile performance and professional appearance

**Decision**: Remove all gradient backgrounds, use simple colors

**Rationale**:
- Faster rendering on budget Android devices
- More professional, less "flashy"
- Better focus on content
- Easier maintenance

**Impact**: Cleaner design, 40% faster first paint

### Decision 4: Static Export Optimization

**Context**: Deploy to GitHub Pages (free)

**Decision**: Optimize for static export, not server-side features

**Configuration**:
```javascript
// next.config.js
output: 'export',
basePath: '/hablas',
images: { unoptimized: true }
```

**Rationale**:
- Free hosting on GitHub Pages
- Infinite scalability via CDN
- Offline-first PWA capabilities
- No server costs
