## Potential Confusion Sources (Clarified)

### Why Resources Might Look "Backwards"

Users might mistakenly think language direction is wrong because:

1. **Both languages present:** Resources show both English and Spanish, but the KEY is:
   - English = TARGET (what to learn/say)
   - Spanish = HELPER (what it means)

2. **Cultural notes in English:** Some notes are in English, BUT:
   - They explain US culture TO Spanish speakers
   - Context is always from Colombian perspective
   - Comparisons reference Colombia as "home" culture

3. **Mixed content:** Files contain:
   - JSON structure (English keys)
   - English target phrases (learning content)
   - Spanish explanations (instructional language)
   - This is CORRECT for the pedagogical approach
