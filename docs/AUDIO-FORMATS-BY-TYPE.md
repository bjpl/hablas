# Audio Format Specifications by Resource Type

## Resource Type Categories:

### 1. **Basic Phrases** (Simple learning)
Resources: 1, 4, 5, 6, 9, 22, 23, 29
Format: Tutorial style
- Intro (30s)
- Per phrase: Context → English 2x (pause 1s) → Spanish → Pause 2.5s for repetition
- Practice section
- Conclusion

### 2. **Conversations/Dialogues** (Interactive practice)
Resources: 10, 16, 20, 32, 33, 34
Format: **DIALOGUE STYLE** - needs adaptation
- Intro explaining conversation practice
- Per exchange: 
  - Setup context
  - Person A speaks → Pause 3s (user repeats/thinks)
  - Person B responds → Pause 3s (user repeats/thinks)
  - Full exchange at normal speed
- Multiple practice rounds
- Conclusion

### 3. **Directions/Numbers** (Quick reference)
Resources: 12, 15, 19
Format: Reference style
- Brief intro
- Each item: English → pause → Spanish → longer pause
- Rapid practice section
- Quick conclusion

### 4. **Emergency/Safety** (Critical info)
Resources: 27, 30, 45-52
Format: Clear instruction style
- Serious tone intro
- Per phrase: Context → English slow 2x → Spanish → LONGER pause (3.5s) for mental preparation
- Emergency numbers
- Conclusion with safety reminders
