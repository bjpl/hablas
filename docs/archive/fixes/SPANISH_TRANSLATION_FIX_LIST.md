## Translation Pattern Reference

### Correct Format

```
[English phrase]
[blank line]
[English phrase - repeat for reinforcement]
[blank line]
[Complete Spanish translation with punctuation]
[blank line]
[blank line]
```

### Common Mistakes to Avoid

❌ **Truncated Spanish** (ends mid-sentence):
```
¿Puede verificar este pedido dos veces?
El cliente tiene alergias. ¿Prepararon    ← INCOMPLETE
```

✅ **Complete Spanish** (proper ending with punctuation):
```
¿Puede verificar este pedido dos veces?
El cliente tiene alergias. ¿Prepararon esto sin ese ingrediente?
```

### Spanish Line Indicators

A proper Spanish line should:
- End with punctuation (. ! ?)
- Not end with prepositions (de, para, con, en, etc.)
- Be a complete thought/sentence
- Include accented characters where appropriate (á, é, í, ó, ú, ñ)
