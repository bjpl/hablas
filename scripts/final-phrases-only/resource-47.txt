LOST ITEM BEST PRACTICES
================================================================================

PREVENTION (Before lost items happen):
1. Remind passengers to check for belongings before exiting
2. Do quick visual sweep after EVERY trip
3. Especially check: seat pockets, floor, between seats

FOUND ITEM PROTOCOL:
1. Take photo of item with timestamp
2. Report through platform app immediately
3. Secure item in separate compartment (not in passenger area)
4. NEVER arrange direct return without platform coordination
5. Keep item secure until platform arranges return

IF ACCUSED OF THEFT:
1. Stay calm and professional
2. Document your empty vehicle with photos/video
3. File platform report immediately with your evidence
4. Request GPS records showing subsequent passengers
5. Do NOT engage in argument - let platform investigate
6. Consider filing police report if accusation escalates

RETURN FEES:
- You ARE entitled to compensation for return trips
- Platform typically offers $15-25 return fee
- You can decline to return if compensation insufficient
- Item can be returned to platform hub instead
